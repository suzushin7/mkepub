# サンプル書籍 (標準版)

これは `mkepub` の動作検証用サンプルMarkdownファイルです。
本サンプルでは、YAMLフロントマターを用いた書籍情報の設定、文章の記述、画像埋め込み、およびコードブロックなどの基本的な機能デモンストレーションを行っています。

# 第1章: はじめに

このツールは、Markdown記法から電子書籍（EPUB）を一瞬で自動生成することを目指して作られました。
昨今、多くの文章がMarkdown形式で執筆されていますが、それを公式な電子書籍フォーマットであるEPUBに変換するには、複雑なオーサリングツールや面倒な設定が必要でした。`mkepub` は、それらの障壁を取り除き、日々の執筆活動をよりスムーズにするためのミニマルなCLIツールです。

- **高速動作**: BunのネイティブファイルIOとJSZipの強力な圧縮ロジックにより、一瞬でEPUBパッケージを作成します。
- **シンプル**: 複雑な設定は一切不要。デフォルトのままでも美しくフォーマットされた電子書籍が手に入ります。
- **拡張性**: 独自のカスタムCSSやカバー画像の設定をサポートし、目次の自動生成も可能です。
- **マルチレイアウト**: 日本語書籍に欠かせない縦書き（rtl）レイアウトにも対応しており、文字送り方向とあわせてフォントや境界線のレイアウトも最適化されます。

## 1.1 読書について

読書は知識を広げ、想像力を豊かにするための素晴らしい手段です。
歴史上の偉大な知性たちと対話し、新しい世界へ旅立つための切符を手に入れることができます。近年では電子書籍の普及により、何百冊もの本を一つの小さなデバイスに入れて持ち歩くことができるようになりました。

![読書](./illustration.png)

技術書であれ、文芸小説であれ、快適なレイアウトで文章を読むことは読書体験そのものの質を大きく左右します。`mkepub` は、その読書体験を最良のものにするため、読みやすさを追求したタイポグラフィの設計をデフォルトCSSに取り入れています。

# 第2章: コードの記述

以下のようにコードブロックも適切に表示されます。
開発者が技術資料やプログラミングの解説書などを執筆する際にも、シンタックスの可読性が保たれるように設計されています。

```javascript
// あいさつを返す簡単な関数
function greet(name) {
  console.log(`Hello, ${name}!`);
}

// 実行例
greet("Bun");
```

コードブロックは、背景に視認性の高いグレーカラーを配し、フォントには等幅フォント（Monospace）を採用しています。これにより、ソースコードのインデントや記号の配置が崩れることなく、美しく整えられて出力されます。また、ダークモード設定時には背景が自動で暗いテーマに切り替わり、目の負担を軽減します。

これで動作確認を行います。
