# 第三章: 自動ビルドの実行

原稿が書き上がったら、最後のステップはEPUBファイルの生成です。

コマンドラインで実行するだけで、一瞬で電子書籍ファイルが完成します。

## 3.1 実行コマンド

連結ビルドを実行するには、以下のように対象ファイルを順番に引数に並べてコマンドを実行します。

```bash
bunx mkepub sample/ch01.md sample/ch02.md sample/ch03.md
```

これにより、最初のファイル（ch01.md）のフロントマターで指定された設定に従って、`sample_combined.epub` という名前の電子書籍が作成されます。
各章の間には自動的に改ページが挿入され、物理的なファイル境界となるため、電子書籍リーダー上でページが美しく切り替わります。
